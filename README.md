## 📌 Overview
This project presents a Sales Performance Dashboard built using Tableau. It provides a consolidated view of sales data, enabling users to analyze key business metrics such as revenue trends, product performance, and regional sales distribution.

The dashboard is designed to support data-driven decision-making by offering intuitive visualizations and interactive filtering capabilities.

## 🎯 Objectives
  - Track overall sales performance over time
  - Identify top-performing products and categories
  - Analyze regional sales distribution
  - Enable quick insights through interactive filters

## 📈 Key Features
- Sales Trends Analysis
    - Visualizes revenue over time to identify growth patterns and seasonality.
    - Category & Product Insights
    - Highlights best- and worst-performing products.
  
- Regional Performance
    - Compares sales across different geographic regions.
  
- Interactive Filters
    - Allows users to drill down into specific segments dynamically.
  
## 🧩 Dashboard Components
Based on the attached presentation , the dashboard includes:
  - KPI summary section (total sales, profit, etc.)
  - Time-series charts for trend analysis
  - Category-wise breakdown
  - Regional comparison visuals
  - Interactive slicers/filters

## 🛠️ Tools & Technologies
